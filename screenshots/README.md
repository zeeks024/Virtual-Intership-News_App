# Screenshots

Folder ini berisi screenshot aplikasi untuk dokumentasi.

## File yang dibutuhkan:
- `home_screen.png` - Screenshot halaman utama dengan list berita
- `detail_screen.png` - Screenshot halaman detail artikel
- `theme_dialog.png` - Screenshot dialog pemilihan tema

## Cara menambahkan screenshot:
1. Ambil screenshot dari HP/emulator
2. Transfer ke komputer
3. Letakkan di folder ini dengan nama file sesuai di atas
4. Commit dan push ke GitHub
